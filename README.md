# 🍔 WhatsApp Delivery Chatbot

Sistema de chatbot para restaurantes que recebem pedidos via WhatsApp e gerenciam tudo em um painel web em tempo real.

---

## 🚀 Funcionalidades

- 📩 Recebimento de pedidos via WhatsApp
- 🧠 Processamento automático de mensagens
- 💾 Armazenamento em banco de dados
- 📊 Painel web para visualização dos pedidos
- 🔄 Atualização de status dos pedidos

---

## 🛠️ Tecnologias

- Node.js
- Baileys (WhatsApp Web API)
- PostgreSQL
- Frontend Web (HTML/CSS/JS ou React)

---

```bash
git clone ...

npm install
npm run dev
